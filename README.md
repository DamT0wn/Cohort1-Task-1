# Gemini ADK Agent – Cloud Run Deployment

## Overview
This project implements a single AI agent using Google ADK and Gemini, deployed on Cloud Run and exposed via an HTTP endpoint.

## Capability
Text Summarization

## Endpoint
POST /summarize

## Sample Request
{
  "text": "Your paragraph here"
}

## Tech Stack
- Vertex AI Gemini
- FastAPI
- Cloud Run
- Python

## Deployment
Built and deployed using Google Cloud Console.
